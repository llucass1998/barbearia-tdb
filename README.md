# TDB Barbearia

Protótipo React/Vite para agenda online da Barbearia Turma do Bairro.

## Rodar

```bash
npm install
npm run api
npm run dev
```

API local: `http://localhost:3334/api`
Site local: `http://localhost:5174`

## Funcionalidades

- Fila por ordem de chegada
- Cadastro rápido de cliente
- Serviços, barbeiros e datas especiais
- Status da fila: aguardando, chamando, em atendimento, finalizado e saiu
- Botão de WhatsApp com mensagem pronta
- Painel admin para controlar a fila
- Backend Express com banco SQLite local
- Lucro/faturamento do dia, semana e mês no admin
